---
title: "AA90: Beneficiary Address Missing"
description: "Resolving issues with bundler's beneficiary configuration"
---

The bundler failed to designate a beneficiary address when processing the user operation.

## Resolution Strategies

1. For Self-Hosted Bundlers:
   - Review your bundler configuration
   - Ensure the beneficiary address is properly set

2. For Third-Party Bundler Services:
   - If using a provider like JiffyLabs, [contact the support team](https://t.me/JiffyScan)

By addressing the beneficiary configuration, you'll resolve the AA90 error and ensure proper fund allocation.
